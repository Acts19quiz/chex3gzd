Links

Here are some links to websites relevant to Chex™ Quest 3 v2, as well as the original games.

CHEX™ 3 GZD GITHUB

This is the official Github page for Chex™ 3 GZD. Here you can find a Wiki offering more detailed information about the engine, dev minutiae, documented version differences, and FAQs. There are also two boards: Issues for researching or submitting your own bug reports; and Discussions for casual Chex™ Quest 3 commentary, observations or opinions.

This is also where the source code is distributed, if you want to delve into the engine's (chex3gzd.exe) internal workings.

https://github.com/Acts19quiz/chex3gzd

https://github.com/Acts19quiz/chex3gzd/issues (Chex™ 3 GZD Issues | bug reports)
https://github.com/Acts19quiz/chex3gzd/discussions (Chex™ 3 GZD Discussions | discuss the game)
https://github.com/Acts19quiz/chex3gzd/wiki (Chex™ 3 GZD Wiki | detailed documentation)

CHARLES JACOBI - GAMER'S PAGE

This is the personal website of Chex™ Quest's original artist, Charles Jacobi, where Chex™ Quest 3 was originally released, and where you can still get the original version 1.4.

http://www.chucktropolis.com/gamers.htm

CHEX QUEST - DOWNLOAD & PLAY OUR CLASSIC GAME - CHEX MIX

This is where www.chexquest.com now redirects, and where you can find information on Chex Quest HD, the high-definition remake of the first five levels of Chex Quest. You can also read the series of comics based on Chex Quest HD.

https://www.chexmix.com/chexquest/

THE ULTIMATE CHEX QUEST

The Ultimate Chex Quest is created by "Boingo the Clown", a big inspiration and influence for the official Chex™ Quest 3. This is an alternate take on and different way to play Chex™ Quest's first ten levels. Note that it is currently at R6 (Release 6), and is still unfinished with minor graphical bugs.

GZDoom 1.3.0 is the minimum required version (1.3.13 or later is recommended for best graphics). g2.2.0 is the absolute latest working version due to serious compatibility issues TUCQ has with newer versions. Zandronum will also work (Options>Set Video Mode>Renderer=OpenGL is recommended). It will run on original ZDoom, however there will be further graphical bugs with it. It is currently not compatible with LZDoom.

http://www.doomlegends.com/chexquest/
